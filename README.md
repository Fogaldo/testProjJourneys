// En version

This project is designed to review front-end and learn the basics of back-end development.

This project created a small landing page for a travel blog.
The project wasn't written entirely by hand. Most of the code was written using DeepSeek AI.
The AI ​​was used as:
   1. Idea generator
   2. Code writing assistant
   3. Mentor for this project

So far, the following has been implemented:
1. Frontend: Loading articles from the server, pagination
2. Backend: Server on Node.js + Express with pagination
3. Counters: Increasing views via API
4. Security: XSS protection via escapeHtml()
5. UI/UX: "Show more" button with centering and loading state
